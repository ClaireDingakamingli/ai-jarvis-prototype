# AI贾维斯 - 智能生活助手原型

## 产品定位
**"你的智能生活伙伴，能主动理解你的生活状态、预判你的潜在需求、并在合适时机提供帮助"**

## 核心创新

1. **从"被动响应"到"主动感知"**
   - 不需要用户输入，自动从生活行为中推理需求
   - 零输入设计，随生活周期变化自动决策

2. **从"单一App"到"多端常驻"**
   - 桌面Widget、语音助手、手机App多端覆盖
   - 实时状态显示，不错过任何提醒

3. **从"购物工具"到"生活伙伴"**
   - 理解生活场景（健身、出差、育儿、家庭）
   - 最短路径解决问题，高置信度自动执行

## 原型展示

本原型包含5个用户角色的20个生活场景：

### 🏋️ 运动健将（蜉蝣）
- 健身房续卡自动识别
- 蛋白粉自动补货
- 运动装备智能推荐
- 跑步路线天气提醒
- 健身计划智能调整

### ⚡ 职场精英（美然）
- 出差装备智能准备
- 社交送礼智能推荐
- 换季衣橱智能分析

### 👶 新手妈妈（欣欣）
- 宝宝用品自动续订
- 宝宝成长阶段提醒
- 宝宝尺码自动升级

### 👨‍👩‍👧 小镇妈妈（丽丽）
- 父母体检智能提醒
- 家庭聚餐智能准备

### 🏠 生活助手
- 健康关怀提醒
- 通勤优化建议

## 技术栈

- React 18
- Lucide React (图标库)
- Tailwind CSS (样式)
- Pollinations.ai (AI图片生成)

## 本地运行

### 前置要求
- Node.js 16+
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 交互说明

1. **选择用户角色**：点击顶部标签切换不同用户视角
2. **查看场景**：点击场景卡片查看详细信息
3. **授权操作**：滑动确认按钮授权AI执行操作
4. **切换视图**：使用底部按钮切换桌面/语音/App视图

## 核心设计原则

- ✅ 零输入（不需要用户配置）
- ✅ 高置信度直接执行（低置信度才询问）
- ✅ 最短路径（能1步完成不要2步）
- ✅ 不依赖日历标记，从自然行为推理
